## Ideia 1: "Swiss Financial Precision"

**Design Movement:** Swiss/International Typographic Style aplicado a dashboards financeiros

**Core Principles:**
1. Grid matemático rigoroso com proporções áureas
2. Hierarquia tipográfica como principal ferramenta de navegação
3. Dados como protagonistas — zero decoração supérflua
4. Contraste cromático cirúrgico para sinalizar saúde financeira

**Color Philosophy:** Paleta monocromática com acentos funcionais. Base em cinza-carvão (#1C1C1E) com superfícies em off-white (#F5F5F0). Verde-esmeralda (#10B981) para valores positivos, vermelho-coral (#EF4444) para negativos, e âmbar (#F59E0B) para alertas. A ausência de cor comunica neutralidade; a presença de cor comunica urgência.

**Layout Paradigm:** Grid assimétrico de 12 colunas com sidebar fixa à esquerda (240px). Conteúdo principal em cards com bordas mínimas e sombras suaves. Tabelas com linhas zebradas e cabeçalhos fixos.

**Signature Elements:**
1. Números financeiros em fonte monoespaçada (JetBrains Mono) com alinhamento decimal
2. Sparklines inline nos KPIs para contexto temporal imediato

**Interaction Philosophy:** Transições rápidas (150ms), hover states com elevação sutil, tooltips informativos em cada métrica.

**Animation:** Contagem animada nos KPIs ao carregar, gráficos que se desenham progressivamente, sidebar que colapsa suavemente.

**Typography System:** DM Sans para títulos e navegação (peso 500-700), Inter para corpo de texto (peso 400-500), JetBrains Mono para valores numéricos e códigos de conta.

## Ideia 2: "Nordic Data Landscape"

**Design Movement:** Design escandinavo minimalista com influência de visualização de dados editorial (estilo Financial Times)

**Core Principles:**
1. Espaço negativo generoso como elemento de design ativo
2. Paleta quente e acolhedora que humaniza dados financeiros frios
3. Tipografia serif para títulos que evoca relatórios impressos de alta qualidade
4. Micro-interações que recompensam a exploração

**Color Philosophy:** Fundo em creme quente (#FAFAF5) com texto em grafite profundo (#2D2D2D). Acentos em azul-petróleo (#0F4C5C) para elementos interativos, terracota (#E07A5F) para destaques e alertas, verde-musgo (#3D5A40) para indicadores positivos. A paleta evoca confiança e sofisticação, distanciando-se do azul corporativo genérico.

**Layout Paradigm:** Layout editorial com navegação superior horizontal e conteúdo em seções full-width com máximo de 1200px. Cards com cantos levemente arredondados (8px) e sombras longas e suaves. Gráficos ocupam largura total com legendas integradas.

**Signature Elements:**
1. Dividers decorativos finos entre seções com gradiente sutil
2. Ícones de linha fina (Lucide) com animação de desenho ao aparecer

**Interaction Philosophy:** Hover revela camadas adicionais de informação. Clique em qualquer métrica abre drill-down contextual. Filtros como chips horizontais deslizáveis.

**Animation:** Fade-in escalonado das seções ao scroll, gráficos com transição morphing ao trocar filtros, números que "rolam" como odômetro.

**Typography System:** Playfair Display para títulos de seção (peso 700), Source Sans 3 para corpo e navegação (peso 400-600), IBM Plex Mono para dados numéricos.

## Ideia 3: "Bauhaus Data Architecture"

**Design Movement:** Bauhaus contemporâneo — formas geométricas primárias como sistema de organização visual

**Core Principles:**
1. Forma segue função — cada elemento visual tem propósito informacional
2. Geometria como linguagem: círculos para proporções, retângulos para comparações, linhas para tendências
3. Cor como código semântico, não decoração
4. Assimetria controlada que guia o olhar pela hierarquia de informação

**Color Philosophy:** Base em branco puro (#FFFFFF) com blocos de cor sólida sem gradientes. Azul-cobalto (#1D4ED8) como cor primária institucional, cinza-aço (#64748B) para elementos secundários, verde-limão (#84CC16) para positivos, rosa-neon (#F43F5E) para negativos. As cores são usadas em blocos geométricos, nunca como fundo difuso.

**Layout Paradigm:** Grid modular com blocos de tamanhos variados (1x1, 2x1, 1x2, 2x2) que se reorganizam responsivamente. Navegação lateral com ícones geométricos. Cada página é uma composição de blocos que pode ser lida como um infográfico.

**Signature Elements:**
1. Blocos coloridos com números grandes (48-72px) como KPI cards
2. Bordas geométricas grossas (3px) em vez de sombras para separação de elementos

**Interaction Philosophy:** Drag-and-drop para reorganizar blocos do dashboard. Clique em blocos expande para detalhamento. Filtros como toggles geométricos.

**Animation:** Blocos que se encaixam como peças de Tetris ao carregar, transições de escala nos hovers, rotação sutil em ícones interativos.

**Typography System:** Space Grotesk para títulos (peso 700), Outfit para corpo (peso 400-500), Fira Code para dados numéricos e códigos.
